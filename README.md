# 项目简介 python966

本项目是一款基于Python的预约管理系统，旨在为教育机构提供一个便捷的课程预约与管理系统。

## 技术栈

- 后端：Python、Django框架
- 前端：JQuery、Bootstrap、HTML、JSP
